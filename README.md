Informatikai projektfeladat 

Egy költés figyelő alkalmazás amiben testreszabhatóan hozhatsz létre füleket ami a költés fejtáját reprezentálja.
A költéseket havi ciklusokra bontva lehet összegezve megnézni, és kimutatásokat látni a korábbi havi költéseihez képest.
Beépítenénk egy API-n keresztül meghívható AI chat botot amivel a költési szokásainkról tudunk beszélni és tanácsokat kérni.

A programot HTML, php és Javascript segítségével hozzuk létre.
