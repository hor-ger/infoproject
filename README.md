Informatikai projektfeladat 

Egy költés figyelő alkalmazás amiben testreszabhatóan hozhatsz létre füleket ami a költés fajtáját reprezentálja.
A költéseket havi ciklusokra bontva lehet összegezve megnézni, és kimutatásokat látni a korábbi havi költéseihez képest. Emellett felhasználná a GVH oldalán lévő jelenleg legfrissebb excel fájlt, amely főként élelmiszerek, háztartási cikkek árát tartalmazza.
Beépítenénk egy API-n keresztül meghívható AI chat botot amivel a költési szokásainkról tudunk beszélni és tanácsokat kérni.

A programot HTML, php és Javascript segítségével hozzuk létre.
